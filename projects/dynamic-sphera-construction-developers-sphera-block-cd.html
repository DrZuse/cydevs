Sphera Block C & D Готов, в продаже

Dynamic Sphera Construction & Developers · Квартиры · Limassol

📷 Фото, планировки и подробности — на странице проекта у застройщика: www.dynamicsphera.com/project/sphera-block-cd

SPHERA BLOCK C & D — два объединённых жилых блока по 3 этажа, всего 16 апартаментов с 1, 2 и 3 спальнями, вид на город и горы. В 5 минутах от главной дороги Лимассола. Проект завершён; в продаже остаётся 1 из 16 юнитов (по данным сайта застройщика).

Проект состоит из 2 объединенных жилых блоков (Block C и Block D). Каждый блок имеет 3 этажа. В Block C на каждом этаже по 3 квартиры: 1-комнатная, 2-комнатная и 3-комнатная. В Block D на первых двух этажах по 2 квартиры (2-комнатные), на третьем этаже Block D расположена 3-комнатная квартира на весь этаж.
