О проекте

Дашборд cydevs — структурированная база застройщиков жилой недвижимости Кипра (виллы, таунхаусы, квартиры, жилые комплексы), их проектов, контактов, отзывов, новостей и судебных дел. Данные хранятся в JSON (data/*.json), дашборд — статический, пересобирается скриптом из данных.

Источники данных

  • Застройщики и проекты: официальные сайты компаний, ассоциация LBDA/ΣΕΤΑΚ (lbda.com.cy), открытые агрегаторы (cyprus-real.estate и др.).
  • Новости: Cyprus Property News, Cyprus Mail, Financial Mirror, In-Cyprus (RSS/HTML).
  • Отзывы: рейтинг и ссылки Google Maps / Trustpilot.
  • Суды и статус компаний: CyLaw (решения), реестр компаний / OpenCorporates (статус).
  • Геолокация: координаты из объявлений и геокодер OpenStreetMap Nominatim.

Ограничения (честно)

  • Крупные порталы (Bazaraki, DOM, home.cy) блокируют автоматический сбор и не имеют публичного API — в автообновление не входят. Часть цен поэтому «по запросу».
  • Полные тексты Google-отзывов бесплатно недоступны (лимит API — 5 отзывов): храним рейтинг и ссылку. Полный сбор включается платным ключом в .env.
  • Активные иски первой инстанции на Кипре не публичны (eJustice закрыт для третьих лиц) — покрываются только опубликованные решения CyLaw, новости и статус компании.
  • Сайты застройщиков могут отдавать 403 скриптам — сбор работает «мягко»: при недоступности источника прежние данные сохраняются, обновление логируется.

Как обновлять

Полный цикл сбора и пересборки: bash update.sh (в среде с доступом в интернет). Автоматически — раз в неделю через GitHub Actions (.github/workflows/update-dashboard.yml), либо запуском вручную. Отдельные шаги — скрипты scripts/collect_*.py.
